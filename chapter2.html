Matrices as System Operators

Matrix Transformations, Matrix Multiplication, Grayscale Images, Rotations, Shearing, Reversibility and Applications in Computer Graphics and Image Processing

Matrix as a System Operator

A matrix acts as a system operator by transforming an input vector into another output vector through matrix multiplication. This makes matrices the fundamental tool for representing linear transformations.

output = A × input
y = A · x

When a matrix A is applied to vector x, it produces a new vector y — potentially with a different magnitude and direction. This is the core idea behind all matrix transformations.

Types of Matrix Transformations

🔄

Rotation

Rotates a vector by a specified angle around the origin

⬛

Scaling

Enlarges or shrinks a vector by multiplying its components

🪞

Reflection

Flips a vector across an axis (x-axis, y-axis, or any line)

↔

Stretching

Elongates a vector along one axis independently of others

◱

Shearing

Slants the shape by shifting one axis relative to another

Rotation Matrix

A 2D rotation matrix rotates every vector in the plane by angle θ (counterclockwise):

R(θ) = [ cos θ −sin θ ]
       [ sin θ cos θ ]

For a 90° rotation: cos 90° = 0, sin 90° = 1, so the matrix maps (x, y) → (−y, x).

Matrix Multiplication

Matrix multiplication is the mechanism by which one transformation is applied after another. The product of two matrices A and B represents the composition of their transformations.

C = A × B
Cᵢⱼ = Σₖ Aᵢₖ × Bₖⱼ

Matrix multiplication is not commutative — A×B ≠ B×A in general. This reflects the fact that doing rotation then scaling gives a different result than scaling then rotating.

Grayscale Images & Image Processing

A grayscale image is a 2D matrix where each entry is a pixel intensity value from 0 (black) to 255 (white). Operations on images are operations on these matrices.

pixel_new = A × pixel_old
  • Brightness adjustment: Add a scalar to every pixel value
  • Contrast enhancement: Multiply the matrix by a scalar
  • Blurring: Apply a convolution matrix (kernel) to the image
  • Edge detection: Use a Laplacian or Sobel operator matrix

Reversibility (Inverse Matrix)

A transformation is reversible if there exists an inverse matrix A⁻¹ such that applying it undoes the original transformation.

A × A⁻¹ = I (identity matrix)
A⁻¹ exists if det(A) ≠ 0

If det(A) = 0, the matrix is singular — it collapses space into a lower dimension and cannot be reversed. Invertibility is crucial in solving linear systems and decoding transformations.
