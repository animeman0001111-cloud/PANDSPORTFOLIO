Chapter 4

Vector Spaces: Degrees of Freedom

Applications in Robotics, RGB Colour Systems, Machine Learning and Computer Science

Introduction to Vector Spaces

Vector spaces form the foundation of Linear Algebra. They are mathematical systems used to represent motion, data, signals, colours, machine learning features and robotic movements.

A vector space allows vectors to be added together and multiplied by scalars while remaining inside the same space. These concepts are widely used in Artificial Intelligence, Computer Graphics, Robotics, Data Science and Engineering.

Vector Space V satisfies: 1. u + v ∈ V (closure under addition) 2. c · v ∈ V (closure under scalar multiplication)

Axioms of a Vector Space

A set V with operations + and · is a vector space over ℝ if all 8 axioms hold:

Closure (+)

u + v is in V for all u, v ∈ V

Commutativity

u + v = v + u

Associativity

(u + v) + w = u + (v + w)

Zero Vector

∃ 0 such that v + 0 = v

Additive Inverse

∃ −v such that v + (−v) = 0

Closure (·)

c · v is in V for all c ∈ ℝ, v ∈ V

Distributivity

c(u + v) = cu + cv

Identity

1 · v = v

Subspaces, Basis and Dimension

A subspace is a subset of a vector space that is itself a vector space under the same operations.

A basis is a set of linearly independent vectors that span the entire vector space. The number of vectors in a basis is the dimension of the space — representing the degrees of freedom.

dim(ℝ³) = 3 → 3 degrees of freedom basis: {[1,0,0], [0,1,0], [0,0,1]}

Applications

🤖 Robotics

A robot arm with 6 joints has 6 degrees of freedom. Each configuration is a vector in a 6D space, enabling precise motion planning.

🎨 RGB Colour Systems

The set of all colours forms a 3D vector space with basis vectors: pure red, pure green, and pure blue.

🧠 Machine Learning

Features of training data form a high-dimensional vector space. PCA finds the subspace that captures the most variance.

💻 Computer Science

Function spaces and signal spaces are vector spaces. Compression algorithms like JPEG operate in transformed vector spaces.
