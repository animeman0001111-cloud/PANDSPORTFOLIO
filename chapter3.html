Chapter 3

Solving System of Linear Equations

Gaussian Elimination, Matrix Methods, LU Factorization and Real World Applications in Engineering, AI and Computer Science

Linear Equation

A linear equation is an equation in which the variables have power one. It represents a straight line when graphed.

ax + by = c

Example: 2x + 3y = 6

Linear equations are the foundation of matrix algebra and computational mathematics. A system of linear equations has multiple equations and unknowns that must be satisfied simultaneously.

a₁x + b₁y = c₁ a₂x + b₂y = c₂

Matrix Form: Ax = b

Any system of linear equations can be written compactly as a matrix equation:

A · x = b [ 2 3 ] [ x ] [ 6 ] [ 1 -1 ] [ y ] = [ 0 ]

Here A is the coefficient matrix, x is the variable vector, and b is the constants vector. Solving this system means finding x = A⁻¹b (when A is invertible).

Gaussian Elimination

Gaussian Elimination is a systematic method for solving systems of linear equations by converting the augmented matrix to row echelon form using elementary row operations.

1

Write the augmented matrix [A | b] combining coefficients and constants.

2

Forward elimination — use row operations to create zeros below each pivot.

3

Back substitution — solve for each variable starting from the last equation upward.

Row operations allowed: swap rows, multiply a row by a scalar, add a multiple of one row to another.

Possible Outcomes

Unique Solution

Exactly one solution exists. Lines intersect at a single point.

No Solution

System is inconsistent. Lines are parallel and never meet.

Infinite Solutions

Lines coincide. Any point on the line is a valid solution.

LU Factorization

LU Factorization decomposes a matrix A into the product of a Lower triangular matrix L and an Upper triangular matrix U:

A = L × U

Once decomposed, the system Ax = b becomes:

L(Ux) = b → Ly = b → Ux = y
  • Solve Ly = b by forward substitution
  • Solve Ux = y by back substitution

LU factorization is highly efficient when solving multiple systems with the same coefficient matrix A but different right-hand sides b.

Real World Applications

  • Engineering: Circuit analysis using Kirchhoff's laws, structural load calculations
  • Computer Graphics: Solving for vertex positions in 3D rendering pipelines
  • AI & Machine Learning: Training linear regression models (minimising least squares)
  • Economics: Input-output models and supply-demand equilibria
  • Robotics: Computing joint angles to achieve a desired end-effector position
