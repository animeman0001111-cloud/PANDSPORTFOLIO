Chapter 1

Data as Vectors

Vector Representation, Matrix Basics, RGB Systems, Spotify Recommendation Systems and Applications of Linear Algebra in AI and Computer Science

What is Data?

Data refers to a collection of numerical or informational values used for analysis, computation and representation. In Linear Algebra, data is represented mathematically using vectors and matrices so that calculations and analysis become easier.

Examples of Data:

  • Student marks
  • Temperature readings
  • Sales records
  • Distances
  • Pixel values
  • Population statistics
  • Machine learning datasets

Vector Representation

A vector is an ordered list of numbers that represents a point or direction in space. Vectors allow us to store and manipulate data mathematically.

v = [v₁, v₂, v₃, ..., vₙ]

1D Vector

A single number: v = [5] representing temperature or price.

2D Vector

A point in a plane: v = [3, 4] representing (x, y) coordinates.

3D Vector

A point in space: v = [1, 2, 3] used in robotics and graphics.

n-D Vector

High-dimensional: used in ML where each feature is one dimension.

Matrix Basics

A matrix is a rectangular array of numbers arranged in rows and columns. It is used to store, transform and analyse data efficiently.

A = [ a₁₁ a₁₂ ]
    [ a₂₁ a₂₂ ]

Key properties: dimensions are written as m × n (rows × columns). Matrices enable us to represent multiple vectors simultaneously and perform batch operations.

RGB Colour Systems

Every colour on a screen is represented as a vector of three values — Red, Green and Blue — each ranging from 0 to 255.

colour = [R, G, B]
Red = [255, 0, 0]
Green = [0, 255, 0]
Blue = [0, 0, 255]
White = [255, 255, 255]

An image is a matrix of RGB vectors. Pixel operations like brightness adjustment and contrast enhancement are linear algebra operations on these matrices.

Spotify Recommendation System

Music streaming platforms represent each song and each user as a high-dimensional vector. Recommendations are found by measuring how similar (close) two vectors are using the dot product or cosine similarity.

similarity = (u · v) / (‖u‖ × ‖v‖)

Songs close in vector space share similar features — tempo, energy, genre — and are likely to appeal to the same listener. This is collaborative filtering powered by Linear Algebra.

Applications in AI & Computer Science

Image Recognition Natural Language Processing Recommendation Engines Data Compression Search Algorithms Computer Vision

Every modern AI model treats input data as vectors. Training neural networks involves repeated matrix operations on these vectors, making Linear Algebra the language of machine intelligence.
